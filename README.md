# Usage
<!-- usage -->
```sh-session
$ npm install -g monoman
$ monoman COMMAND
running command...
$ monoman (-v|--version|version)
monoman/0.0.0 linux-x64 node-v14.16.1
$ monoman --help [COMMAND]
USAGE
  $ monoman COMMAND
...
```
